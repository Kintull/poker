# poker

This is a test application that help you with comparing poker hands.

Run application:

> mix compile

> iex -S mix

> Poker.start

Exit application:
<Ctrl-G>, q


Test input 1:
	Black: 2H 3D 5S 9C KD White: 2C 3H 4S 8C AH
Result 1:
	White wins - high_card: Ace


Test input 2:
	User1: 3H 3D 3S 9C KD User2: 2C 3H 4S 5C 6H
Result 2:
	User2 wins - straight
